## Новые возможности

### Приоритеты задач

- Возможность установки приоритета задачи (низкий, обычный, высокий, срочный)
- Визуальное отображение приоритетов в интерфейсе

### Сроки выполнения

- Установка срока выполнения задачи
- Автоматическое определение просроченных задач
- Визуальное выделение просроченных задач

### Уведомления

- Система уведомлений при назначении задач
- Уведомления об изменении назначения задач
- Уведомления о приближающихся сроках выполнения задач
- Email-уведомления для важных событий

### Экспорт данных

- Возможность экспорта задач в формат CSV
- Экспорт учитывает примененные фильтры

### Расширенные фильтры

- Фильтрация задач по статусу
- Фильтрация по дате создания
- Фильтрация по пользователю (для администраторов)

### Автоматические уведомления о сроках

Для автоматической отправки уведомлений о приближающихся сроках выполнения задач, добавьте в cron следующую команду:

```bash
# Проверять ежедневно в 9:00 уведомления о сроках выполнения задач
0 9 * * * cd /path/to/project && php bin/console app:send-deadline-notifications
```

Замените `/path/to/project` на реальный путь к вашему проекту.

### Регистрация и восстановление пароля

- Страница регистрации: `/register`
- Страница восстановления пароля: `/forgot-password`
- Страница сброса пароля: `/reset-password/reset/{token}`
- Пользователи могут зарегистрироваться самостоятельно
- Восстановление пароля через email с одноразовой ссылкой
- Защита от частых попыток сброса пароля
