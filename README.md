# 🚜 CRM система анализа продаж

**ООО «Дальневосточный фермер»** — современная CRM-система для агробизнеса с расширенной аналитикой, автоматизацией продаж и управлением клиентскими отношениями.

## 📊 О проекте

Полнофункциональная CRM-система на базе Symfony 8.0, разработанная специально для предприятий агробизнеса. Предоставляет комплексные инструменты для управления продажами, анализа эффективности и автоматизации бизнес-процессов.

### ✨ Ключевые возможности

#### 📈 Управление продажами
- **Сделки (Deals)** — воронка продаж, этапы сделок, история изменений
- **Клиенты (Clients)** — полная CRM-база, взаимодействия, история контактов
- **Продукты (Products)** — каталог товаров, учёт продаж, аналитика
- **Бюджетирование** — планирование и контроль финансовых показателей

#### 📋 Управление задачами
- **Задачи** — создание, назначение, приоритеты, дедлайны, зависимости
- **Канбан-доска** — визуальное управление с drag & drop
- **Календарь** — планирование с экспортом в iCal
- **Шаблоны задач** — автоматизация повторяющихся процессов

#### 📊 Аналитика и отчётность
- **Дашборды** — KPI, метрики продаж, визуализация данных
- **Отчёты по продажам** — динамика, прогнозы, тренды
- **Анализ клиентов** — сегментация, отток, LTV
- **Товарная аналитика** — оборачиваемость, маржинальность
- **Эффективность менеджеров** — рейтинг, конверсия, выполнение плана

#### 🤖 Автоматизация
- **AI Ассистент** — умные подсказки, прогнозирование
- **Автоматизация задач** — триггеры, правила, сценарии
- **Уведомления** — real-time, email, push, настраиваемые алерты
- **Вебхуки** — интеграция с внешними системами

#### 👥 Командная работа
- **Ролевая модель** — User, Analyst, Manager, Admin
- **Комментарии и упоминания** — обсуждение задач и сделок
- **Документы** — хранение и управление файлами
- **База знаний** — статьи, категории, внутренний wiki

#### 🎯 Дополнительные модули
- **Геймификация** — достижения, уровни, таблица лидеров
- **Учёт времени** — трекинг, Pomodoro, отчёты по времени
- **Цели (Goals)** — постановка и отслеживание целей компании
- **Привычки (Habits)** — формирование полезных практик
- **Аудит** — логирование действий, безопасность

## 🎯 Целевая аудитория

- **Руководители** — контроль показателей, планирование, отчётность
- **Менеджеры по продажам** — управление сделками, клиентами, задачами
- **Аналитики** — глубокая аналитика, дашборды, прогнозирование
- **Сотрудники агробизнеса** — учёт, планирование, координация

## 🚀 Технологии

- **Backend:** PHP 8.5, Symfony 8.0, API Platform
- **Frontend:** Stimulus, Turbo, Bootstrap 5, FullCalendar, Chart.js
- **Database:** PostgreSQL 16+ / MySQL 8.0+
- **Cache:** Redis (опционально), Symfony Cache
- **Queue:** Symfony Messenger
- **Architecture:** DDD, CQRS, Event-Driven Architecture
- **Security:** Symfony Security, 2FA, CSP, RBAC

### Установка

```bash
# 1. Клонирование репозитория
git clone https://github.com/your-repo/to-do-list.git
cd to-do-list

# 2. Установка зависимостей
composer install
npm install

# 3. Настройка окружения
cp .env .env.local
# Отредактируйте .env.local с настройками БД и почтового сервера

# 4. Создание базы данных и миграции
php bin/console doctrine:database:create
php bin/console doctrine:migrations:migrate

# 5. Создание тестовых данных
php bin/console app:create-test-data

# 6. Сборка assets
npm run build

# 7. Запуск сервера разработки
symfony server:start
# или
php -S 127.0.0.1:8080 -t public
```

Откройте браузер: http://127.0.0.1:8080
