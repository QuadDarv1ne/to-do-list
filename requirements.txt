# Application Requirements

## Software Requirements
- PHP 8.1 or higher
- Composer (PHP dependency manager)
- SQLite3 or MySQL database
- Web server (Apache/Nginx) or built-in PHP server

## PHP Extensions Required
- pdo_sqlite (for SQLite support)
- pdo_mysql (for MySQL support)
- json
- ctype
- iconv
- mbstring
- xml
- zip
- openssl
- tokenizer
- xmlwriter
- intl

## Frontend Requirements
- Bootstrap 5.x
- Font Awesome 6.x
- JavaScript ES6+ compatible browser

## System Requirements
- Minimum 2GB RAM (4GB recommended)
- 100MB free disk space (for application files)
- Internet connection (for initial setup and package downloads)
