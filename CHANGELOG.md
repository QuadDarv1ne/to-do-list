# Changelog

Все заметные изменения в проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
версионирование следует [Semantic Versioning](https://semver.org/lang/ru/).

## [Не выпущено]

### Добавлено
- EditorConfig для консистентности кода
- PHP CS Fixer для автоматического форматирования (PSR-12)
- PHPStan для статического анализа (уровень 5)
- Pre-commit git hook для автоматической проверки
- Документация стандартов кода (CODE_STANDARDS.md)
- План улучшений проекта (IMPROVEMENT_PLAN.md)
- Composer скрипты: `cs`, `cs:fix`, `phpstan`, `test`, `check`
- Git attributes для контроля EOL
- Unit тесты для TimeTrackingService
- Unit тесты для Task entity

### Изменено
- Расширен .editorconfig (94 строки правил)
- Обновлён README.md с инструкциями по проверке кода
- Обновлён composer.json с dev-зависимостями

### Исправлено
- Конфигурация PHP CS Fixer (убраны несуществующие правила)
- Конфигурация PHPStan (совместимость с версией 2.x)
