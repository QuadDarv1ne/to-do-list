СИСТЕМА ТЕМ - БЫСТРАЯ ИНСТРУКЦИЯ
==================================

Реализовано 5 тем оформления:

1. СВЕТЛАЯ (light) - классическая светлая тема
2. ТЁМНАЯ (dark) - снижает нагрузку на глаза
3. ОРАНЖЕВАЯ (orange) - энергичная и яркая
4. ФИОЛЕТОВАЯ (purple) - креативная и стильная
5. НАСТРАИВАЕМАЯ (custom) - создайте свою цветовую схему

КАК ИСПОЛЬЗОВАТЬ:
-----------------
1. Нажмите на иконку палитры в навбаре (справа вверху)
2. Выберите нужную тему из модального окна
3. Для настраиваемой темы - выберите цвета и нажмите "Применить"
4. Включите автопереключение для смены темы по времени суток

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
-------------------
- Темы сохраняются в localStorage
- Плавные переходы между темами
- Поддержка системных предпочтений (prefers-color-scheme)
- Автоматическое переключение день/ночь (опционально)
