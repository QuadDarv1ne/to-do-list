# Требования к приложению

## Программное обеспечение
- PHP 8.1 или выше
- Composer (менеджер зависимостей PHP)
- SQLite3 или MySQL база данных
- Веб-сервер (Apache/Nginx) или встроенный PHP сервер

## Необходимые расширения PHP
- pdo_sqlite (для поддержки SQLite)
- pdo_mysql (для поддержки MySQL)
- json
- ctype
- iconv
- mbstring
- xml
- zip
- openssl
- tokenizer
- xmlwriter
- intl

## Требования к frontend
- Bootstrap 5.x
- Font Awesome 6.x
- Совместимый с JavaScript ES6+ браузер

## Системные требования
- Минимум 2 ГБ ОЗУ (рекомендуется 4 ГБ)
- 100 МБ свободного места на диске (для файлов приложения)
- Подключение к Интернету (для первоначальной настройки и загрузки пакетов)
