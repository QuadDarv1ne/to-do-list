## Статистика по модулям

### 1. Система задач (11 сущностей)

- Task - основная сущность задач
- TaskCategory - категории задач
- TaskTag - теги задач
- TaskComment - комментарии к задачам
- TaskTimeTracking - учет времени
- TaskRecurrence - повторяющиеся задачи
- TaskTemplate - шаблоны задач
- TaskDependency - зависимости задач
- TaskHistory - история изменений
- TaskNotification - уведомления по задачам
- TaskAttachment - вложения к задачам

### 2. Пользователи и безопасность (7 сущностей)

- User - основная сущность пользователей

- UserDashboardLayout - настройки дашборда
- UserDevice - устройства пользователей
- UserIntegration - интеграции с внешними сервисами
- ResetPasswordRequest - сброс пароля
- ActivityLog - лог активности

### 3. CRM-система (4 сущности)

- Client - клиенты
- Deal - сделки
- ClientInteraction - взаимодействия с клиентами
- DealHistory - история сделок

### 4. Финансовая система (3 сущности)

- Budget - бюджеты
- Product - продукты и услуги
- Document - документы

### 5. Система уведомлений (4 сущности)

- Notification - уведомления
- NotificationPreference - настройки уведомлений
- NotificationTemplate - шаблоны уведомлений
- Webhook - вебхуки

### 6. Система целей и привычек (4 сущности)

- Goal - цели
- GoalMilestone - этапы целей
- Habit - привычки
- HabitLog - лог привычек

### 7. Система знаний (2 сущности)

- KnowledgeBaseArticle - статьи базы знаний
- KnowledgeBaseCategory - категории статей

### 8. Системные сущности (11 сущностей)

- Category - общие категории
- Tag - общие теги
- Comment - общие комментарии
- Resource - ресурсы
- ResourceAllocation - распределение ресурсов
- FilterView - сохраненные фильтры
- SavedSearch - сохраненные поиски
- Skill - навыки
- WebhookLog - лог вебхуков
- Mention - упоминания
- EventStore - хранилище событий
