### 3. ✅ Мета-теги OpenGraph

**Primary Meta Tags:**

**Twitter Card:**
- `twitter:card` — тип карточки (summary_large_image)
- `twitter:url` — URL
- `twitter:title` — заголовок
- `twitter:description` — описание
- `twitter:image` — изображение
- `twitter:creator` — автор (@dvfarm)

**Результат:** Красивые превью ссылок в соцсетях (VK, Telegram, Facebook, Twitter)

### 5. ✅ Health Check Endpoint

**Созданный файл:**
- `/src/Controller/HealthCheckController.php`

**Доступные эндпоинты:**

| Endpoint | Описание | Использование |
|----------|----------|---------------|
| `GET /health` | Базовая проверка (liveness) | Load balancer |
| `GET /health/ready` | Полная проверка (readiness) | Kubernetes, мониторинг |
| `GET /health/live` | Проверка живости | Orchestrators |
| `GET /health/version` | Информация о версии | Отладка |

**Пример ответа `/health`:**
```json
{
  "status": "healthy",
  "timestamp": "2026-03-10T12:00:00+07:00",
  "version": "3.2.0",
  "environment": "dev"
}
```

**Пример ответа `/health/ready`:**
```json
{
  "status": "ready",
  "timestamp": "2026-03-10T12:00:00+07:00",
  "version": "3.2.0",
  "environment": "dev",
  "checks": {
    "database": {
      "status": "healthy",
      "message": "Database connection successful"
    },
    "cache": {
      "status": "healthy",
      "message": "Cache directory is writable"
    },
    "logs": {
      "status": "healthy",
      "message": "Log directory is writable"
    }
  }
}
```
