# Содержание папки отчёт_для_ТУСУР

## Документация по базе данных системы управления задачами

### Основные файлы:

1. **описание_базы_данных.md** - Полное описание структуры базы данных
   - Общая информация о системе
   - Технические характеристики
   - Подробное описание всех 46 сущностей
   - Связи между сущностями
   - Индексы и оптимизация
   - Меры безопасности

2. **er_диаграмма.md** - ER-диаграмма системы
   - Визуальное представление структуры БД
   - Все связи между сущностями
   - Легенда диаграммы
   - Цветовая кодировка
   - Кратность связей

3. **техническое_задание.md** - Техническое задание на разработку
   - Функциональные требования
   - Нефункциональные требования
   - Архитектурные требования
   - План разработки
   - Критерии приемки

4. **отчет_по_практике.md** - Отчет по производственной практике
   - Общая характеристика проекта
   - Выполненная работа
   - Результаты и достижения
   - Используемые технологии
   - Решенные задачи
   - Приобретенные навыки

5. **статистика_базы_данных.md** - Статистическая информация
   - Количественные показатели
   - Статистика по модулям
   - Информация о связях
   - Индексация и оптимизация
   - Производительность

## Структура проекта:

```
отчёт_для_ТУСУР/
├── описание_базы_данных.md
├── er_диаграмма.md

├── отчет_по_практике.md
├── статистика_базы_данных.md
└── содержание.txt
```
